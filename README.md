# project-template
Template for a reproducible data analysis project using RStudio, R Markdown, Quarto, and GitHub. The structure below gives an example of files for the individual folders/direcories. Each folder/directory contains a README.md file with additional information.

- data
    - processed
        - cleaned_data.csv
        - processed_data.rds
    - raw
        - original_data.csv
        - spreadsheet_data.xlsx
- output
    - figures
        - 01_figure.png
        - 02_figure.pdf
    - tables
        - 01_table.csv
        - 02_table.rds
- products
    - manuscript
        - manuscript.docx
        - manuscript.html
        - manuscript.pdf
        - manuscript.qmd
    - report
        - report.html
        - report.qmd
    - slides
        - slides.html
        - slides.qmd
- R
    - 00_main.R
    - 01_cleaning.R
    - 02_analysis.R
    - 03_plotting.R
- .gitignore
- projectname.Rproj
- README.md
